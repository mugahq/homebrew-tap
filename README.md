# Muga Homebrew Tap

Official Homebrew tap for [Muga](https://muga.sh) — observability for your services.

## Install

```sh
brew install mugahq/tap/muga
```

## Upgrade

```sh
brew upgrade muga
```

## Usage

```sh
muga --help
```

## Other installation methods

See the [releases page](https://github.com/mugahq/muga/releases) for direct binary downloads (Linux, macOS, Windows).
